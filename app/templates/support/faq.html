It is free.

There isn't a version for install via the Apple or Google App Stores. However, it is a PWA so you can install this application onto your mobile phone by:

  • Android click the install button on the Welcome page. This will create a shortcut for the website on your phone and enable you to quick launch it without opening a browser.
  • Apple whilst using the website, click the Share button on Safari and you will have the option to share with the homescreen. This will create a shortcut directly onto your home screen so you can launch logmyexercise without opening a browser.
